## Features:
>
> - takes character or franchise name from filename  
(supports zerochan, danbooru, yande.re)
> - recognizes other sources too  
(supports X, reddit, others get marked as "no source")
> - has "primary tag" feature:  
if art has only one character or title it'll and be primary (hello zerochan)  
> - has advanced search suggestion system, and you can exclude tags from results too!  
> - image fullscreen mode - see image better, see tags too and adjust them  
> - convenient tag management page for advanced adjustments  
> - and many more!

## Installation:  
>
> you'll need:  
> - git
> - visual studio 2022/2026 to have C++ something something package (trust me bro)
> - python (i have 3.12)  
> - imagesize libary  
> - flet 0.84.3  
> - flutter 3.41.3 (flet will install it on it's own)  
>
> how to:  
> - download code  
> - unzip archive  
> - Step 3!: in `{whatever}\yumemigusa\yumemigusa folder` run `flet build windows` command  
> - `.exe` is in `{whatever}\yumemigusa\yumemigusa\build\windows\yumemigusa.exe`
